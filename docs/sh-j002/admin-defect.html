    動画をドラッグ&ドロップで GitHub リポジトリにアップロードするには GitHub PAT (Personal Access Token) が必要です。 この設定はブラウザに保存され、サーバーには送信されません。

    为了通过拖拽将视频上传到 GitHub 仓库，需要 GitHub PAT (个人访问令牌)。 此设置保存在浏览器本地，不会发送到任何服务器。

    1. github.com/settings/personal-access-tokens/new を開く
    2. Repository access → Only select repositories で次の3つを選択:
      shojiki-rakuten-stats / shojiki-defects-j001 / shojiki-defects-j002
    3. Permissions → Contents: Read and write と Issues: Read and write を ON
    4. 「Generate token」→ 表示された github_pat_... を全コピーして下に貼り付け
