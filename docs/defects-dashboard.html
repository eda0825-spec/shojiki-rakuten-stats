⚙️ アグリゲートビューを有効化する手順 / 启用聚合视图

このページで両 private リポの Issue 一覧を直接表示するには、サーバー経由の同期が必要です。手順:

  1. GitHub Settings → Developer settings → Personal access tokens で repo スコープの PAT を発行
  2. shojiki-rakuten-stats リポの Secrets に DEFECTS_SYNC_PAT として追加
  3. 追加済みのワークフロー sync-defects.yml が日次で両リポの Issue を JSON にダンプし、docs/data/defects-*.json として commit
  4. 本ページがその JSON を読んで集約表示

PAT 投入が済めば、次のスケジュール実行 (毎日 09:00 JST) でこの集約セクションが自動表示される。
